# 🎹 Virtual Piano - Casio CT-X870IN Style
## ✅ Features
### 🎹 **Keyboard Controls with Visible Labels**
Each piano key displays the corresponding computer keyboard letter!
#### **White Keys (C - C)**
- **A** = C
- **S** = D
- **D** = E
- **F** = F
- **G** = G
- **H** = A
- **J** = B
- **K** = C (next octave)

#### **Black Keys (Sharps)**
- **W** = C#
- **E** = D#
- **T** = F#
- **Y** = G#
- **U** = A#

### 🎚️ **5 Octave Selection**
Switch between 5 different octaves using:
- Press **1** for Octave 1 (lowest)
- Press **2** for Octave 2 (default)
- Press **3** for Octave 3 (middle)
- Press **4** for Octave 4
- Press **5** for Octave 5 (highest)

Or click the octave buttons at the bottom of the piano!

### 🎨 **Visual Features**
- **Keyboard letters visible on each key** - No need to memorize!
- **Active octave indicator** - Shows which octave you're playing
- **Highlighted keys** - Keys light up when pressed
- **Modern Casio CT-X870IN design**
- **Smooth animations and transitions**
- **Speaker pulse effects**
- **LED visualizer**

### 🎵 **Sound Features**
- **10 different voices**: Grand Piano, Bright Piano, E.Piano, Church Organ, Guitar, Violin, Synth Lead, Saxophone, Vibes, Bass
- **Reverb effect** - Adds spacious ambience
- **Echo effect** - Adds rhythmic delay
- **Master volume control**
- **Realistic ADSR envelope** for each voice

## 🎮 How to Use

1. **Open the piano**: Open `index.html` in your web browser
2. **Click anywhere** to enable audio (browser requirement)
3. **Select an octave**: Click buttons 1-5 or press number keys 1-5
4. **Play with keyboard**: 
   - Use **A, S, D, F, G, H, J, K** for white keys
   - Use **W, E, T, Y, U** for black keys
5. **Change sounds**: Click VOICE button, then use +/- to browse voices
6. **Add effects**: Toggle REV (reverb) and ECHO buttons
7. **Adjust volume**: Use the MASTER VOL slider

## 💡 Tips

- **The keyboard letters are shown on each piano key!** Look at the piano to see which computer key to press
- **Hold keys** for sustained notes
- **Try different octaves** - Lower octaves for bass, higher for treble
- **Experiment with voices** - Each has unique characteristics
- **Use effects** - Reverb adds space, Echo adds rhythm
- **Play chords** - Press multiple keys simultaneously
